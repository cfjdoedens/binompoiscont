# Package index

## stable

These functions are considered stable.

- [`dbinom_continuous()`](https://cfjdoedens.github.io/binompoiscont/reference/dbinom_continuous.md)
  : Continuous version of dbinom(k, n, p, log = FALSE)
- [`dpois_continuous()`](https://cfjdoedens.github.io/binompoiscont/reference/dpois_continuous.md)
  : Poisson density function, version continuous in both arguments
- [`dpois3_continuous()`](https://cfjdoedens.github.io/binompoiscont/reference/dpois3_continuous.md)
  : Poisson density function, version continuous in both arguments
